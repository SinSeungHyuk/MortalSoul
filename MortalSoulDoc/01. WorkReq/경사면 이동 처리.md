# 경사면 이동 처리

## 개요
타일맵에 약 45도 경사면이 있을 때 플레이어가 올라가지 못하는 문제 수정.

## 원인 분석
현재 `PlayerMovementController.FixedUpdate`에서 `rb.linearVelocity = new Vector2(curVelocityX, rb.linearVelocityY)`로 수평 속도만 설정하고 있음. 경사면에서는 수평 이동만으로는 경사를 타고 올라갈 수 없음.

## 작업 내용

### 1. 경사면 법선(Normal) 감지
- `UpdateGroundCheck`에서 기존 BoxCast의 hit 결과로부터 `hit.normal`을 저장
- 지면에 닿아 있을 때 경사면의 기울기 정보를 얻음

### 2. 경사면 따라 이동 방향 투영
- 지상 이동 시 `Vector2.Perpendicular(groundNormal)`을 이용하여 경사면 접선 방향 계산
- `curVelocityX` 대신 경사면 접선 방향으로 속도를 투영하여 적용
- 공중(점프/낙하)에서는 기존 수평 이동 유지

### 3. 경사면 미끄러짐 방지
- 지면에 정지 시 경사면에서 미끄러지지 않도록 중력 보정 또는 velocity.y = 0 처리
- 필요 시 PhysicsMaterial2D(friction=0)를 플레이어 콜라이더에 적용하여 마찰 제거

### 4. 최대 등반 각도 제한 (선택)
- 너무 급한 경사(예: 60도 이상)는 올라가지 못하도록 제한하는 상수를 `Settings`에 추가 검토
